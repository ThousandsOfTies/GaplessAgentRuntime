## 仮想 H/W 操作

人間も AI も入口は `gar sim ...` に統一。接続先 EC2 host は `gar setup` で `.gar/config.json` に保存されるため `--host` は省略可。

```bash
# GPIO17 ボタンを短押し
gar sim ui button press 17 --duration-ms 150

# ボタン状態を直接セット（0=離す / 1=押す）
gar sim ui button set 17 1

# RFID カードをタップ / 外す
gar sim ui rfid tap 04:AB:CD:EF:01:23
gar sim ui rfid remove

# VL53L0X 距離センサー値を変更
gar sim ui range set 450

# 仮想 H/W の現在状態（OLED framebuf 含む）を取得
gar sim env status --json

# port forward と bridge の共有状態を確認
gar sim env status
```
