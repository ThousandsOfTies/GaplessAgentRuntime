## 基本方針

| 人間向け | AI エージェント向け |
|---|---|
| Web Panel で LED / ボタン / RFID / OLED を見る | `gar sim ui button/rfid/range` と `gar sim env status --json` で仮想 H/W を操作・観察する |
| SSH でログを見る | `gar sim env log` / `gar sim env diag`（AI は `gar sim env diag --json`）で観察する |
| 手順書を読みながら実行する | `gar` コマンドと Make ターゲットを組み合わせて実行する |

AIに任せたい操作は、なるべく「短く、明示的で、再実行しやすいコマンド」にします。
