## EC2 シミュレータ操作

`gar sim` の接続先 EC2 host は `gar setup` で `.gar/config.json` に保存します。

```bash
# gar-build-env Codespace でビルドし、成果物を WSL hub 経由で EC2 へ転送

# EC2 上でテスト用 /dev/* runtime を起動
# Hardware Panel 用の 8080/8765 port forward も同時に開始
gar sim env start

# VS Code terminal profile "EC2 Simulation" などから EC2 へ入り、
# 本番と同じアプリ起動手順を実行
~/sensor_demo

# ログ確認
gar sim env log

# プロセス、デバイス、API状態をまとめて確認
gar sim env diag

# 代表操作を実行
gar sim ui button press 17
gar sim ui rfid tap 04:AB:CD:EF:01:23
gar sim env status --json

# 停止
gar sim env stop
```
