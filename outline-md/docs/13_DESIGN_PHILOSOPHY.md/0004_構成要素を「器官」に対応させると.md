## 構成要素を「器官」に対応させると

| 器官 | Gapless Agent Runtime の要素 |
|---|---|
| 想像力 / 内部世界モデル | 仮想デバイス（CUSE / gpio-sim / bridge） |
| 感覚・自己認識 | `gar sim env diag --json` の `ok` / exit code |
| 本能・行動規範 | `AGENT.md` の AI オペレーション契約 |
| 運動を実行する小脳 | AI（Codex / Copilot） |
| 意志（何をしたいか） | 人間 |

足りないピース: **仮想での予行結果を見て、実機に出す前に AI が自分で「やめておく / 直す」
判断をする閉ループ。** ここが繋がると、Gapless Agent Runtime は「指示されて動くツール」から
「試してから動く何か」へ変わる。
